# Person Grouped Card Feature

**Status**: Planning Complete - Ready for Implementation  
**Created**: 2025-01-15  
**Feature Type**: New Frontend Card

## Overview

Mobile-first card combining person points display with person-switchable grouped task view. Users can tap the person header to reveal a dropdown selector and quickly switch between different people's tasks.
