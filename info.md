# ChoreBot Cards

Custom Lovelace cards for the ChoreBot integration.

## Cards Included

### ChoreBot List Card
Displays tasks from a ChoreBot todo entity with enhanced features:
- Filter by tags
- Filter by date
- Filter by completion status
- Special effects on task completion (confetti)

### ChoreBot Add Button Card
A button card that opens a dialog for adding new tasks with custom fields:
- Summary and description
- Tags
- Recurrence rules (rrule)
- Due dates

## Installation

This plugin requires the ChoreBot custom integration to be installed first.
